Assignment 06: Gesture Control Game

Overview

For this assignment, I created a simple gesture-controlled game using webcam input. I was inspired by classic arcade games but wanted to make it more interactive by using motion detection. The basic idea is that players can control the game character by waving their hands, which I thought would be more fun than just using keyboard controls.

Design Process

I chose a guava-like pink and avocado green color scheme because I wanted something that looked relaxing and warm. The main challenge was figuring out how to accurately detect different hand movements. I also added the "clear all obstacles" special move to give players a way to clear the screen, but has a cooldown for it. Besides that, I added a camera toggle button so players can choose between seeing the webcam feed or just enjoying the gradient background with falling snow effect.
